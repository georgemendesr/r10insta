📹 OVERLAY DE CAPA DE VÍDEO

Para adicionar um overlay personalizado para as capas de vídeo:

1. Crie uma imagem PNG com dimensões 1080x1920 pixels (vertical)
2. Use transparência (canal alpha) onde o conteúdo deve aparecer
3. Salve o arquivo como: overlay-video.png
4. Coloque neste diretório (templates/)

⚠️ IMPORTANTE:
- O overlay será aplicado SOBRE a imagem de fundo
- Use transparência para permitir que a imagem de fundo seja visível
- O texto do título será renderizado por cima do overlay

🎨 DICAS DE DESIGN:
- Deixe espaço no centro para o título
- Use elementos decorativos nas bordas
- Mantenha consistência visual com o branding R10

Até que você adicione o overlay personalizado, o sistema funcionará
normalmente, apenas sem o overlay decorativo.
